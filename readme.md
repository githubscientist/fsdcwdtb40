# TailwindCSS

- TailwindCSS is a utility-first CSS framework for rapidly building custom user interfaces.

Library:

- Collection of classes, methods, or functions that are pre-written to perform specific tasks and can be reused across different projects.

Framework:

- Framework is a collection of libraries and tools that provide a structured way to build and organize applications, often enforcing certain design patterns and conventions.

CDN:

Content Delivery Network is a system of distributed servers that deliver web content to users based on their geographic location, improving load times and performance.
